認識概念

什麼是 Vibe Coding？

一種全新的程式開發方式——你負責提想法，AI 負責寫程式

01// 你說：幫我做一個待辦清單
02// AI 自動生成：
03
04function addTask(taskName) {
05  const task = {
06    id: Date.now(),
07    name: taskName,
08    done: false
09  };
10  tasks.push(task);
11}

不會寫程式？沒關係！

傳統學程式需要幾個月甚至幾年，但 Vibe Coding 讓你跳過死背語法的過程，直接用「說話」的方式指揮 AI 寫程式。

你就像一位導演，AI 是你的工程師——你描述想要的畫面，AI 幫你實現。

💡 Vibe Coding 的精髓：
用你懂的話，做出你想要的東西。

入門教學

從零開始的第一步

跟著這幾個步驟，你今天就可以做出第一個作品

1

選一個工具，建立帳號

新手建議從 Claude 或 Bolt.new 開始，兩個都不用安裝任何東西，在瀏覽器就能直接用，而且有免費版。

2

想清楚你想做什麼

不需要很複雜的技術術語，用日常語言描述你的需求。越具體越好，例如顏色、功能、使用對象都可以說。

❌ 太模糊
「幫我做一個網頁」
✅ 描述清楚
「幫我做一個個人名片網頁，背景用深色，顯示我的名字、職業是設計師、聯絡信箱，風格要簡約有質感」
3

把你的需求輸入給 AI

直接把描述貼進對話框，送出後 AI 會開始生成程式碼。如果結果不滿意，直接繼續說「把背景改成藍色」或「加一個聯絡按鈕」。

💡 小技巧
可以一直修改！把 AI 當成你的助手，持續對話直到滿意為止。
4

執行並預覽結果

在 Bolt.new 或 Replit 裡，程式碼可以直接在旁邊預覽。如果是在 Claude 或 ChatGPT，可以把生成的 HTML 存成檔案，用瀏覽器打開就能看到成果。

5

繼續優化，做出你想要的樣子

把對話框當成你的修改工具。看到什麼不對勁，直接說出來，AI 幫你改。慢慡疊加功能，你的作品就會越來越完整。

常用的修改指令
「幫我加一個深色模式切換按鈕」

「讓這個頁面在手機上也能看」

「把字體改大一點，顏色用橘色」

常見問題

你可能想知道的事

我完全不懂程式，真的可以學嗎？ +
可以！Vibe Coding 的核心就是「不需要懂程式也能做出程式」。你只需要清楚地描述你想要什麼，剩下的交給 AI。當然，做的過程中你也會慢慢學到一些東西，但這不是必要條件。
AI 生成的程式碼，我看不懂也沒關係嗎？ +
完全沒關係！你可以讓 AI 解釋每一段程式碼的意思，或者直接繼續描述你想修改的地方，不需要自己去改程式碼。不過，慢慢了解一些基礎概念會讓你用起來更順手。
用 Vibe Coding 能做出什麼等級的東西？ +
個人網頁、工具型網站、自動化腳本、簡單的 App 都可以做。複雜的企業級系統目前還是需要有經驗的工程師，但對大多數個人或小團隊的需求來說，Vibe Coding 已經非常夠用。
要花多少錢？ +
推薦的工具都有免費版，初學者完全可以從免費版開始。如果使用量大或需要進階功能，升級付費版大約每月 $10–$20 美元不等。
AI 說的話都是對的嗎？ +
AI 也會犯錯！生成的程式碼有時會有 bug，或是沒有完全理解你的需求。這很正常——遇到問題時，把錯誤訊息或不對的地方告訴 AI，它通常可以幫你修好。這個來回修正的過程，其實就是學習的一部分。
